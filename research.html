Embedded Object Detection For Drones

Cornell 2019-20

Recent advances in deep learning has resulted in applications like object detection. Object detection models are computationally heavy and hence, some research focus has shifted from increasing accuracy to deploying these models efficiently on embedded devices. To give an impetus for research in this direction the Design Automation Conference (DAC) has organised an annual contest System Design Contest (SDC) for detecting objects from aerial videos captured using drones. The contest is judged on a combination of localization accuracy, frame rate, and energy usage. The objective of this project is to develop a computationally light object detection model by performing an effective model search without compromising on accuracy, latency and power usage. For testing the model the competition organizers have collabaroted with Xilinx and provided us with Xilinx Ultra96-V2 containing Xilinx Zynq UltraScale+ MPSoC ZU3EG A484. Project Advisor- Prof. Zhiru Zhang

Machine Learning for Garbage Collection in Datacenters

Cornell Spring'20

The cost of garbage collection in a datacenter varies widely during the application’s execution. Therefore, it is very important to decide during which time the garbage collection must take place in order to minimize the cost. Choosing these points in time at which garbage collection must take place is critical. Our ultimate goal is to develop an algorithm that will dynamically determine the best times to garbage collect on datacenter machines. We note that different systems like Spark and Cassandra must have differing garbage collection methods, which highlights the need for communication and reconfigurability. We identify two important aspects of this system as using the right policy and collecting at the right time. Project Advisor- Prof. Christina Delimitrou

Community Detection in Social Networks

Cornell Spring'20

Community detection is a very popular and challenging problem in modern network science. There are various algorithms for detecting communities in a network, depending on the definition of a community. There are various algorithms like modularity maximization, community detection by statistical inference, and normalized-cut graph partitioning. This project aims to implement these algorithms on a real social network dataset and try analyse the strengths and the weakness of these algorithms. Project Advisor- Prof. Vikram Krishnamurthy

Deep Learning Approach to Video Compression

Sardar Patel Institute of Technology, 2018-19

Video streaming requirement has increased exponentially and video currently consumes 75% of the internet traffic. Due to which video streaming and storage is a huge challenge for service providers. Image and video compression algorithms rely on codecs which are encoders and decoders that lack adaptability. Due to the advent and advances in Deep Learning these issues can be solved. This paper proposes a method for video compression using neural networks that outperforms the H.264/AVC video coding standard as measured using Multi-Scale - Structural Similarity Index (MS-SSIM).The neural network model proposed is a multi-layer architecture consisting of two parts i) Encoder and ii) Decoder. The training of the two parts of the model happens together and during test time the encoder and decoder are separated to be used as just like any other compression encoding and decoding modules. The entire model’s purpose was to try and capitalize on the temporal and spatial dependencies between frames of a video.

e-Yantra- IIT Bombay Robotics Contest

IIT Bombay, 2017-18

Contributed to the team’s 1st place finish in e-Yantra-17 conducted by Indian Institute of Technology, Bombay. The theme of this competition was constructing two robots called 'Collector Bot'and 'Truck' marked with Aruco markes, that perform fruit collection; where 'fruits' were thermecol cubes also marked using Aruco markers so that an overhead camera can detect them. I worked on motion planning, path planning, OpenCV image processing, PID algorithm in Python using the software V-REP, and microcontroller programming for the robots. The objective was to construct a robot that would collect and deposit fruits, avoiding obstacles using an efficient algorithm in the shortest possible time.

e-Nirogya- An IoT Healthcare based Start-up

Cornell Spring'20

e-Nirogya is a structured system incorporating a health diagnosis machine that measures several body parameters and an iot-based platform for its operation. e-Nirogya also connects a network of medical practitioners/hospitals with rural India. Our mission was to eradicate the lack of health care facilities in rural India by creating a chain of primary health care units that offer high quality, affordable outpatient care in low income, densely populated regions, where health care delivery by the state is sparse. Contributed to the team’s 1st place finish in 'Interthrone IoT startup competition' for the startup e-Nirogya.
